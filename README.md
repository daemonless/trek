## Deployment

### Podman Compose

```yaml
services:
  trek:
    image: ghcr.io/daemonless/trek:latest
    container_name: trek
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=UTC
    volumes:
      - "/path/to/containers/trek:/config"
    ports:
      - 3000:3000
    restart: unless-stopped
```
